# APP TO LOAD DATA AND RENDER IT IN CARDS

# STACK
1. Nuxt3
2. Vue3
3. TypeScript
4. Pinia
5. TailwindCss

# IDE
VsCode

# SETUP

Install the dependencies:
# yarn
```yarn install```
# npm
```npm install```


## Development Server

Start the development server on http://localhost:3000

```
npm run dev
```
